## Описание проекта
API для управления задачами с использованием матрицы Эйзенхауэра. Позволяет создавать, просматривать, фильтровать и искать задачи по различным критериям.

## Технологии
- **FastAPI** - веб-фреймворк для создания API
- **Python 3.13+** - язык программирования
- **Uvicorn** - ASGI сервер для запуска приложения
- **Pydantic** - валидация данных

## Матрица Эйзенхауэра
- **Q1**: Важно + Срочно (Кризисы, дедлайны)
- **Q2**: Важно + Не срочно (Планирование, развитие)
- **Q3**: Не важно + Срочно (Отвлечения, рутина)
- **Q4**: Не важно + Не срочно ("Воры" времени)

## Доступные эндпоинты

### Основные
- `GET /` - Информация об API
- `GET /tasks` - Все задачи
- `GET /tasks/stats` - Статистика по задачам

### Фильтрация
- `GET /tasks/{task_id}` - Задача по ID
- `GET /tasks/quadrant/{quadrant}` - Задачи по квадранту (Q1-Q4)
- `GET /tasks/status/{status}` - Задачи по статусу (completed/pending)

### Поиск
- `GET /tasks/search?q={query}` - Поиск по названию/описанию
